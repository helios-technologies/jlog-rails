jlog-rails
==========

JLog - Rails integration with server side logger for JLog.js

Usage:
------

In the Gemfile:

    gem 'jlog-rails'

then:

    rails g jlog:install

On the client-side:

    var logger = new JLog();
    logger.addAppender(new JLog.AjaxAppender('/jlog/append'));
    logger.warn('warning to be recorded in the server-side log');
